Hadoop Pseudo-Distributed Setup

偽分布式模式(Pseudo-Distributed Mode)

  • 在單個集群上運行Hadoop
  • 所有的Hadoop daemon皆運行在不同的Java process
  • 在單機模式上增加了程式調適功能，並允許檢查以下狀態:
    • 記憶體使用情況
    • HDFS的輸出/輸入
    • 與其他daemon的交互情況，如namenode，datanode，secondarynamenode，jobtracer，tasktrace
